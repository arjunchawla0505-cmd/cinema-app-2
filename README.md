🎬 CineSearch — Movie Explorer

A cinematic movie search app built with HTML, CSS & JavaScript. No API key required.

Features


Live search suggestions as you type
Movie poster, plot, genres, runtime, release year
Director, cast, language, country & box office
Direct IMDb page link
Fully responsive


Tech


Pure HTML / CSS / JS — no frameworks
Wikipedia & Wikidata APIs (free, no key needed)
Hosted on GitHub Pages
